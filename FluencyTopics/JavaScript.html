Loops

In this example with loops were using the for loop
with the for loop we can tell the loop to do it a certain amount of times.
in this example 35 times

function loopexample(){
var birthday = "";
for (i = 0; i < 36; i++) {
birthday += i;
}
document.getElementById("loopdemo").innerHTML = birthday;
}
